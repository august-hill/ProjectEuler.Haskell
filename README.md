# Project Euler Solutions - Haskell

100 [Project Euler](https://projecteuler.net/) solutions in Haskell, part of a cross-language benchmark suite with standardized benchmarking across 11 languages.

See [ProjectEuler.Benchmarks](https://github.com/august-hill/ProjectEuler.Benchmarks) for cross-language performance comparisons.

## Prerequisites

- **Compiler**: GHC (Glasgow Haskell Compiler)
- **Platform**: macOS / Linux

## Running

```bash
# Run a single solution
cd problem_001
ghc -O2 -o main_bench Main.hs -i..
./main_bench

# Run all benchmarks (from repo root)
./run_all.sh
```

## Project Structure

```
ProjectEuler.Haskell/
  Bench.hs                  # Shared benchmarking module
  problem_001/Main.hs       # Solution + benchmark for problem 1
  problem_002/Main.hs
  ...
  problem_100/Main.hs
```

Each `Main.hs` imports `Bench` and outputs timing results in a standardized format consumed by the Benchmarks aggregator.
